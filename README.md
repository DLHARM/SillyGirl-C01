# SillyGirl-C01

本项目是一个开源手机项目，这个项目主要用于深入学习构建安卓系统、linux系统、鸿蒙系统所搭建的移动硬件平台，更多资料作者还在整理完善之中，欢迎志同道合的小伙伴一起学习交流

## 硬件规格

| 组件        | 参数描述                     |
|-------------|----------------------------|
| 主控芯片     | Rockchip RK3568 (4xCortex-A55) |
| GPU         | Mali-G52                   |
| 内存         | 2GB/4GB LPDDR4X            |
| 存储         | 64GB/128GB eMMC            |
| 显示屏       | 5.99" AMOLED 1920X2160      |
| 通信模块     | 4G LTE + WiFi6 + BT5.0      |
